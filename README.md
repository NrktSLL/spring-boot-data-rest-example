# Spring Boot Data Rest Example 

> It is an example of Data Rest with Hal Explorer
>

## Used Dependencies
* Spring Boot Data Rest 
* Spring Boot Validation
* Spring Boot Hateoas
* Spring Boot Data JPA
* H2 Database 
* HAL Explorer
* Lombok
* Dev Tools

## HAL Explorer
> **Access : http://localhost:8080/api/**
